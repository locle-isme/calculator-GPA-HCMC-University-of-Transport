# CALCULATOR GPA BACKEND

## Environment

[NodeJS](https://nodejs.org/en/)

## Project setup

Run terminal line to install necessary packages in package.json:

### For yarn:

```bash
yarn install
```

### For npm:

```bash
npm install
```

## Building

Add new except courses in variable `listBanned` :

```javascript
let listBanned = [
    '010100700704',
    '010100700604',
    '010100410121',
    '010100700804',
    '010100410505',
    '010100410601',
    '010100410802',
];
```

Run terminal line:

```bash
node main.js
```
